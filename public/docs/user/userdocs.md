# Wiki - User Guide

The Wiki plugin is your personal and shared knowledge space. Create pages to save notes, study summaries, or anything else useful — and publish pages to share them with others in your community.

## Creating Pages

Click **New Page** at the top of the page tree to create a new root-level page.

To create a **subpage** (a page nested under an existing one), right-click any page in the tree and select **Add subpage**.

Each page has:

- **Icon** — An emoji or symbol shown in the tree (for example: 📖, ⚙, ★)
- **Title** — The page name shown in the tree and as a heading
- **Description** — An optional subtitle shown below the title
- **Content** — The main body, written in markdown

## Editing Pages

Click the **edit button** (pencil icon) on any page to open the editor. The editor toolbar lets you format your content:

- **Text**: Bold, italic, strikethrough, inline code
- **Structure**: Paragraphs, headings (H1–H3)
- **Lists**: Bullet lists, numbered lists
- **Blocks**: Code blocks, blockquotes

Click **Save** when done, or **Cancel** to discard your changes.

## Organizing Pages

Pages are arranged in a tree structure in the left sidebar. You can:

- **Create subpages** — right-click a page and select "Add subpage"
- **Move a page** — right-click a page and select "Move", then choose a new parent
- **Reorder pages** — set the sort order in the editor to control the position among siblings
- **Delete a page** — right-click a page and select "Delete"

The **breadcrumb trail** at the top of each page shows the full path from the root to the current page. Click any breadcrumb to navigate up.

## Private vs. Public Pages

All pages you create start as **private** — only you can see them.

To **publish a page** (make it visible to everyone):
- Right-click the page in the tree and select **Publish**
- Or click the globe icon on the page viewer

To make a published page private again, right-click it and select **Unpublish**.

Use the **My Pages** and **Public** tabs at the top of the sidebar to switch between your private pages and all published pages.

## Comments and Reactions

Below each page there is a comments section where you can:

- **Write a comment** — type in the text area and click "Comment"
- **React with an emoji** — click an existing reaction to add yours, or click "+" to choose from: 👍 ❤️ 😄 🎉 🤔 👀
- **Delete your own comments** — use the trash icon next to your comment

## Sidebar Access

The Wiki is also available from the side panel. Open the side panel and click the Wiki icon to browse pages without leaving the current plugin. Click any page to navigate to it in the main wiki view.

## Study Plan Integration

Pages can be tagged with an **action label** in the editor. When tagged, the Study Plan plugin can assign the page as an exercise. When assigned, you are navigated directly to that page.

## Tips

- Use the right-click menu on pages for quick actions: add subpage, move, delete, publish, or unpublish.
- Give top-level pages meaningful icons so they are easier to spot at a glance.
- Use subpages to organize related notes under a single topic.
